# GPMO Mascot + Asset Design Brief

## Project Overview
Gentoo Penguin Math Organization (GPMO) is a youth-focused competition math community. We need a cohesive mascot and visual asset system that feels energetic, smart, and welcoming for middle/high school math competitors.

## Brand Personality
- Energetic, playful, and confident
- Competitive but friendly
- Clean and modern, not childish
- Academic vibe with momentum and fun

## Existing Assets
- Current logo: assets/images/penguins/logo.png
- Mascot base reference: assets/images/penguins/noot.png
- Theme icons already in use: assets/images/icons/sun.svg and assets/images/icons/moon.svg

## Design Goals
1. Build a mascot kit based on the noot penguin.
2. Create reusable icon/badge assets for contests and timeline milestones.
3. Ensure all key assets work in both light and dark UI themes.
4. Keep visuals scalable and web-optimized.

## Required Deliverables

### 1) Mascot Kit (based on noot.png)
- Neutral standing pose
- Trophy/celebration pose
- Thinking/studying pose
- Team/collaboration pose
- Coming soon/announcement pose
- Wave/hello pose

For each pose, provide:
- Full color light-mode version
- Full color dark-mode-safe version (or transparent/outlined variant)
- Monochrome single-color version

### 2) Contest Badge Set
- GPMC 10/12 badge
- GPIME badge
- GPMT badge

Each badge should support:
- Small UI usage (timeline, cards)
- Medium usage (section headers, posters)

### 3) Utility Icons
- Timeline marker icon
- Resources/download icon style set
- Discord/community icon style that matches brand

### 4) Social + Marketing Graphics
- Open Graph social preview image (for link sharing)
- Discord banner option (optional but preferred)
- Homepage hero illustration option using mascot + math motifs

## Visual Direction
- Keep the penguin expressive and recognizable in silhouette.
- Use geometric math motifs (grid, compass arcs, coordinate hints, clean equation symbols) as secondary elements.
- Avoid clutter; prioritize strong shape language and readability on mobile.

## Color Direction
Use a warm/cool dual-accent system that harmonizes with current UI:
- Warm accent: red/rose family
- Cool accent: cyan/teal/sky family
- Neutral support for both light and dark backgrounds

## Technical Specs
- Preferred source format: SVG (vector)
- Large artwork export: WebP (plus transparent PNG fallback if needed)
- Provide 1x and 2x raster exports where relevant
- Keep files optimized for web (small file size, clean paths)

## Naming + Organization
Suggested file structure:
- assets/images/penguins/mascot-[pose]-[variant].svg
- assets/images/icons/[name].svg
- assets/images/badges/[contest]-badge-[variant].svg
- assets/images/social/og-gpmo-main.webp

## Usage Constraints
- Assets must remain clear when scaled down for nav/cards.
- Must look good on both light and dark themes.
- Avoid copyrighted third-party graphics or fonts.

## Success Criteria
- Mascot feels unique to GPMO and instantly recognizable.
- Badge/icon set feels unified and consistent with current site style.
- Assets improve homepage/contest page personality without hurting readability.
- Final exports are production-ready for direct drop-in.

## Notes for Designer
- The noot penguin should be the base identity anchor.
- Preserve friendly expression and avoid over-rendered realism.
- Prioritize clean vector forms and strong contrast.
